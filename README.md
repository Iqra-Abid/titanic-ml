# Titanic Survival Prediction 

Built an ML model to predict Titanic survival using Random Forest.

## Results
- Baseline accuracy: 0.753
- After feature engineering: 0.811
- After hyperparameter tuning: 0.841

## What I used
- Python, pandas, scikit-learn
- Feature engineering (Title, FamilySize, IsAlone)
- GridSearchCV for tuning
